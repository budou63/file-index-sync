# 新ファイル基準表 説明書

## 差分インポートCSV作成マクロ

実行するマクロ名: `手順4_差分インポートCSVを作成する`

### 全体の考え方

この作業では、まず新システムから出力したCSVをもとに `新ファイル基準表` を作成します。

利用者は `新ファイル基準表` を編集し、修正したい内容を反映します。その後、このマクロで `目録CSV` と `新ファイル基準表` を比較し、変更された部分だけを `差分インポートCSV` として抽出します。

作成した `差分インポートCSV` は総務へ渡し、新システムへインポートしてもらいます。インポートが完了すると、新システム側のデータにも編集内容が反映されます。

つまり、最終的には `新ファイル基準表` の内容と、新システムから再出力したCSVの内容が一致する状態を目指します。

このマクロは、Excel上で編集した内容を、新システム側へ戻すための橋渡しをする役割です。

### 目的

このマクロは、`目録CSV` と `新ファイル基準表` を比較し、変更があったデータだけを抽出して `差分インポートCSV` を作成するためのマクロです。

作成した `差分インポートCSV` は、新システムへ更新データとして取り込むことを想定しています。

### 処理の流れ

1. `目録CSV` を読み込む
2. `新ファイル基準表` を読み込む
3. 同じフォルダかどうかを照合する
4. 内容が変更されているか確認する
5. 変更がある行だけを抽出する
6. `差分インポートCSV` を作成する

### 同じフォルダかどうかの照合条件

次の項目をもとに、同じフォルダかどうかを判定します。

| 照合項目 | 内容 |
|---|---|
| 年度（和暦） | R7、R8など |
| タイトル | フォルダ名 |
| 分類名2 | 第1ガイド相当 |
| 媒体種別 | 紙、電子など |
| 保存期間 | 1年、3年、5年など |

年度が違う同名フォルダは、別データとして扱います。

### 差分として抽出されるもの

次のような変更がある場合、差分として抽出されます。

- タイトル変更
- 分類変更
- 保存期間変更
- 色変更
- 媒体種別変更
- その他の編集内容

### 差分対象外となるデータ

次のデータは、差分対象外になります。

| 条件 | 理由 |
|---|---|
| 備考に「マクロ差込」または「差込マクロ」と記載されている | 旧ファイル基準表から補助的に追加したデータのため |
| 表記ゆれのみで、実質的に同じ内容 | 誤差分を防ぐため |

### 表記ゆれの吸収

次のような違いは、同じ内容として扱います。

- 全角と半角の違い
- 括弧の違い
- 空白の違い
- 改行の違い
- 一部記号の違い

例:

```text
（2工区）
(2工区)
```

上記は、同じ内容として判定されます。

### 注意事項

- `目録CSV` は元データのため、直接編集しないでください。
- 編集は `新ファイル基準表` で行ってください。
- `差分インポートCSV` には、変更があるデータのみ出力されます。
- 年度違いの同名フォルダは、別データとして扱われます。

### このマクロの役割

このマクロは、必要な変更データだけを安全に新システムへ戻すためのマクロです。

不要なデータや誤差分をできるだけ除外し、必要な更新だけを抽出することを目的としています。
